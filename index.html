我们是谁

《银杏新评》是一份由学生个人投稿文章组成的文学评论集。我们希望收集和整理来自学生个人的真实经验、公共观察、时事评论与学术思考。这里不是正式社团，也不是公开组织，而是一处用于承载文字的空间。

我们为什么要做

校园里外，师生上下，都有很多令人困惑或不满的事情，也有很多人有学术方面的思考。我们知道，有人沉默，但是总有人想发声。

为什么不是表白墙

表白墙适合短消息和临时表达，但并不适合作为文章的发表地方。因此，我们设立《银杏新评》，供所有想发出声音的写作者使用。

我们的运营方式

1. 接受投稿

来自个人或群体，内容不限，长度不限。无论是吐槽还是学术文章都欢迎。

2. 简单审核

投稿内容会经过简单审核，主要核查是否违反法律红线，并排版整理。

3. 整理发布

审核通过后，文章将被收录进此网页。希望大家转发，让更多人看见。

投稿方式

目前《银杏新评》采用 QQ 投稿方式。请加入我们的 QQ 群，或直接联系运营账号发送稿件。建议格式：文章分类 + 标题 + 作者/笔名 + 正文。

投稿须知

1. 内容主题限制较少，字数不限。
2. 文章内容和论述应基于事实或明确的个人经历，不捏造、不夸大。
3. 可实名、匿名或笔名投稿，投稿者身份不会被对外公布。
4. 稿件一经录用，即视为授权《银杏新评》进行非营利性复制与存档，作者保留署名权。

我们是谁

《银杏新评》是一份由学生个人投稿文章组成的文学评论集。我们希望收集和整理来自学生个人的真实经验、公共观察、时事评论与学术思考。这里不是正式社团，也不是公开组织，而是一处用于承载文字的空间。

我们为什么要做

校园里外，师生上下，都有很多令人困惑或不满的事情，也有很多人有学术方面的思考。我们知道，有人沉默，但是总有人想发声。

为什么不是表白墙

表白墙适合短消息和临时表达，但并不适合作为文章的发表地方。因此，我们设立《银杏新评》，供所有想发出声音的写作者使用。

我们的运营方式

1. 接受投稿

来自个人或群体，内容不限，长度不限。无论是吐槽还是学术文章都欢迎。

2. 简单审核

投稿内容会经过简单审核，主要核查是否违反法律红线，并排版整理。

3. 整理发布

审核通过后，文章将被收录进此网页。希望大家转发，让更多人看见。

投稿方式

目前《银杏新评》采用 QQ 投稿方式。请加入我们的 QQ 群，或直接联系运营账号发送稿件。建议格式：文章分类 + 标题 + 作者/笔名 + 正文。
